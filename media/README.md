# Media Assets

This folder contains logos, images, and video assets for x402Resolve.

## Files

- `kamiyo_logomark.svg` - KAMIYO logo (primary)
- `icon.jpg` - Favicon for website
- `kamiyo.jpeg` - KAMIYO logo (alternate format)
- `Phantom-Icon_Transparent_White.svg` - Phantom wallet icon
- `phantom.png` - Phantom wallet icon (alternate format)
- `demo-video.mp4` - Project demo video (placeholder)

## Usage

All media assets are referenced from the root `/media/` path in the repository.

For GitHub Pages, use: `https://x402kamiyo.github.io/x402resolve/media/[filename]`
